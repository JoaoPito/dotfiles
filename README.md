These are the dotfiles for my setup.
It mainly uses Catppuccin for the theming (**Mocha** for dark mode and **Latte** for light mode)

## Files
**Go to "~/.config/":**
- Hyprland
- waybar
- kitty
- wofi/rofi
- mako
- neovim

**Go to "~":**
- ZSH

**Go to "/usr/share/":**:w
- sddm (Change the theme in the file /etc/sddm.conf.d/10-theme.conf)

**Others:**
- Wallpapers
